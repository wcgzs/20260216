# 安全事件管理平台（纯前端）

这是一个纯前端 Web 应用，可用于网络安全事件的增删改查与多维统计。

## 功能

- 安全事件新增、查询、编辑、删除（CRUD）
- 告警管理（新增、编辑、删除、多选）
- 支持将多条告警合并并提交为一个安全事件
- 多条件筛选（关键词、严重等级、状态、类型、负责人）
- 多维统计（按严重等级、状态、类型、来源）
- 本地 `localStorage` 持久化

---

## 能否在你自己的电脑本地访问？

可以，完全可以。

这个项目没有后端依赖，下载代码后可以直接在本地运行。

### 方式一：直接打开（最简单）

1. 下载或克隆项目到本地。
2. 双击 `index.html`，用浏览器打开。

> 说明：在大多数浏览器中可直接使用。

### 方式二：本地启动静态服务器（推荐）

在项目目录执行以下任一命令：

#### 2.1 使用 Python（若已安装）

```bash
python3 -m http.server 4173
```

然后浏览器访问：

- `http://localhost:4173`

#### 2.2 使用 Node.js（若已安装）

```bash
npx serve . -l 4173
```

然后浏览器访问：

- `http://localhost:4173`

---

## 数据说明

- 所有事件数据保存在浏览器本地：`localStorage`。
- 清空浏览器站点数据后，事件数据会被清除。

## 项目结构

- `index.html`：页面结构
- `styles.css`：页面样式
- `app.js`：业务逻辑（CRUD、筛选、统计、持久化）
